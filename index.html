Hi, I'm Mohammed. Since starting my journey in Computer Science in 2022, I’ve spent my time deep in the architecture of algorithms, APIs, and databases. I’ve grown to understand the "invisible" side of technology—the complex logic that keeps the digital world running.

Throughout my studies at AASTMT, I’ve noticed a recurring problem: The more robust the code, the harder it is to explain. To a client or stakeholder, your most brilliant architecture is often just a "black box." They can't see the innovation happening under the hood.

06 // Visualized Logic

Featured Work

Translating backend complexity into frontend clarity through high-fidelity motion.

System Architecture

SQL Request & JSON Response Visualized

A clean, schematic visualization of a standard client-database interaction. This piece illustrates the direct communication between an application and its database, tracking a raw SQL query (SELECT * FROM users) across the network. It highlights the return trip, animating the data stream as individual JSON objects that are retrieved and ultimately rendered into a structured table within the application's UI.

Tooling: After Effects, Illustrator
Algorithm Visualization

Pixel Reduction: Image Compression

A breakdown of how compression algorithms optimize file sizes without destroying visual fidelity. This animation visualizes the process of scanning a pixel grid, identifying redundant color data, and clustering similar pixels together. It bridges the gap between the mathematical logic of data reduction and the visual result seen by the end user.

Tooling: After Effects, Illustrator
Data Structures

LIFO Logic: The Stack Explained

Demystifying one of the most fundamental data structures in computer science. This loop illustrates the Last-In, First-Out (LIFO) principle in action. By visualizing memory allocation, it clearly demonstrates how "push" and "pop" operations manipulate the execution context, giving a physical sense of weight and order to backend memory management.

Tooling: After Effects, Illustrator
